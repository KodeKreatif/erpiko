Requirements :
A. Ubuntu on Windows then install the following packages :
  1. git
  2. curl
  3. cmake
  4. autoconf
  5. automake
  6. libtool
B. Microsoft Visual Studio 2015 with C++ compiler


Run install.bat to build the dependency
Run build.bat to build erpiko
Run test.bat to test erpiko

var.bat contains information about platform (x86 or x64) and configuration (Debug or Release)
